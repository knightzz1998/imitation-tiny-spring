### 解决Github连接问题

参考 https://zhuanlan.zhihu.com/p/481574024

```shell
# 使用socks5代理（推荐）
git config --global http.https://github.com.proxy socks5://127.0.0.1:51837
```

- 51837 是 socks5的端口号
